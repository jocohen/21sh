BUILTINS:

	ENV:
		- cf man env -> standard POSIX:
			- options: -i (initiate env = NULL with name=value)
			- arg env: if name=value add it to env just for one
			- if no utility print it
			- utility:
				- if no path (no '/'), take PATH
			- return:
				- 0 on success
				- 127 utility not be found (env -i 21sh => no path so error 127)
				- 126 utility found but can't be executed
				- >0 error

	CD:
		if no option, works like -L
		-options:
				(ex: link->Folder)
			- L: go to link
			- P: go to Folder
			- -: go to OLDPWD
			if -L -P or -LP or -PL or -P -L get -P option
		- If no directory (Path) go to $HOME (if HOME doesn't exist error "cd:
		  HOME not set")
		- as always when start by / or . ..
		- return:
			- 0 on success
			- >0 error
